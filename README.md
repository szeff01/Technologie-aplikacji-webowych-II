# Instrukcja Uruchomienia projektu

## Uruchomienie backendu

1. Przejdź do folderu backend:
   ```bash
   cd backend
   ```
2. Zainstaluj zależności:
   ```bash
   npm install
   ```
3. Uruchom server backendu:
   ```bash
   npm start
   ```

## Uruchomienie frontendu

1. Przejdź do folderu frontend:
   ```bash
   cd frontend
   ```
2. Zainstaluj zależności:
   ```bash
   npm install
   ```
3. Uruchom frontend:
   ```bash
   npm run dev
   ```
